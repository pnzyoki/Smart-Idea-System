# ✨ Smart Idea System

A responsive React + Vite workspace for capturing, scoring, and organizing your ideas. Built with a premium dark aesthetic, AI-powered grouping, and a dual-mode architecture that works entirely offline with no setup required.

---

## Features

- **Idea Capture** — Submit ideas with dynamic Impact, Feasibility, and Effort sliders that score and classify your idea in real time
- **AI Co-Pilot** — Semantic idea clustering and per-card critique drawers with score audits and actionable improvement bullets
- **Analytics Dashboard** — Track momentum, streaks, quadrant distribution, and category breakdowns
- **Offline Ready** — Runs fully in-browser using localStorage with zero configuration needed

---

## Local Development

**Prerequisites**: Node.js 18+

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```

Open `http://localhost:5173` in your browser. Use any email and password (min. 6 characters) to sign in to the local sandbox — no account setup required.
